# Green Verify

An intelligent preassessment tool for green building certification that predicts GRIHA ratings for buildings using machine learning and AI.

## Features

- Predict GRIHA ratings (1-5 stars) based on building parameters
- AI-powered chatbot (GreenyBot) for sustainability guidance
- Modern web interface with real-time predictions
- Detailed building assessments and recommendations

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/green-verify.git
   cd green-verify
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Set up environment variables (optional for AI features):
   - Create a `.env` file
   - Add `GEMINI_API_KEY=your_api_key_here`

4. Run the application:
   ```bash
   python app.py
   ```

5. Open your browser and go to `http://localhost:5000`

## Usage

1. Enter building parameters in the input form
2. Click "Analyze Building" to get your GRIHA rating prediction
3. View results and get AI-powered assessments
4. Chat with GreenyBot for expert advice
